Задание:
Реализовать веб-интерфейс и клиентскую логику функциональности «список контактов».

Функциональность:
1) При первом заходе на страницу список с существующими контактами загружается аяксом из json-файла (можно использовать любой файл-заглушку, не нужно писать сервер) и сохраняется в localStorage
2) При последующих посещениях страницы, список с контактами берется из localStorage
3) Пользователь может добавить новый контакт к существующему списку (название и адрес - обязательные для заполнения поля), который сохраняется в localStorage
4) Пользователь может осуществлять поиск по существующим контактам, вводя название организации
5) Пользователь может переключать вид отображения со списка на карту и обратно.

Условия:
1) Реализуйте функциональность и обеспечьте кроссбраузерность (все популярные браузеры, последние версии).
2) Реализуйте корректное отображение интерфейса на планшетах и мобильных устройствах (адаптивный интерфейс)
3) В реализации решения вы можете использовать любые технологии, которые демонстрируют ваши знания и навыки.
4) Результатом выполнения тестового задания должно быть хорошо написанное с точки зрения Кандидата решение, протестированное, удовлетворяющее исходному описанию, имеющее максимально аккуратный и чистый код
